# Cloud Usage & Cost Tracker

## Description
This project tracks AWS cloud usage and estimates monthly costs using Excel.
It focuses on EC2 and S3 services under AWS Free Tier.

## Features
- Tracks EC2 uptime and S3 storage usage
- Calculates monthly cost using formulas
- Visualizes resource-wise spending
- Identifies idle and unused resources
- Includes basic test cases for validation

## Tools Used
- Microsoft Excel
- AWS Free Tier

## Use Case
Helpful for beginners to understand cloud cost management and monitoring.
